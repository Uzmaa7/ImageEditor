# Image Editor

A **web-based image editor** that allows users to apply multiple real-time filters to images, reset them, use predefined presets, and download the edited image. Built with **HTML, CSS, and Vanilla JavaScript**, this project demonstrates dynamic DOM manipulation, canvas rendering, and modern UI design.

This project enables users to enhance their images directly in the browser. Features include:

- **Upload & Preview Images:** Upload any image and see it rendered on a `<canvas>` element.
- **Real-time Filters:** Adjust:
  - Brightness
  - Contrast
  - Saturation
  - Hue Rotation
  - Blur
  - Grayscale
  - Sepia
  - Opacity
  - Invert
- **Presets:** Apply predefined filter combinations such as `Vintage`, `Black & White`, and so on.
- **Reset Filters:** Restore all filter values to their defaults.
- **Download Image:** Save the edited image as a PNG file.
- **Responsive & Modern Design:** Uses CSS variables, flex layouts, and intuitive UI elements.
